# 🧠 AI-Powered Customer Feedback Intelligence System

An end-to-end NLP system that analyzes Amazon Fine Food Reviews 
to extract actionable customer insights using AI.

## 🔗 Live Demo
[View Dashboard](YOUR_STREAMLIT_URL_HERE)

## 📊 Features
- Sentiment Analysis using RoBERTa (97% accuracy)
- Topic Modeling using BERTopic
- Keyword Extraction using KeyBERT
- Interactive Streamlit Dashboard
- 568,000+ reviews analyzed

## 🛠️ Tech Stack
- Python, Pandas, HuggingFace Transformers
- BERTopic, KeyBERT, SQLite
- Streamlit, Plotly, WordCloud

## 📁 Dataset
Amazon Fine Food Reviews — 568,454 reviews from Kaggle

## 🚀 How to Run Locally
pip install -r requirements.txt
streamlit run dashboard.py
